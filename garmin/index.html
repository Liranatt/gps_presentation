Statistical Analysis

Patterns in the Data

Beyond the daily numbers, what relationships and patterns emerge from 48 days of continuous health monitoring? The following sections use established statistical methods to uncover meaningful structure in the data.

Correlation Analysis

How Do Health Metrics Move Together?

Pearson correlation measures whether two health metrics tend to move in the same direction (positive correlation), opposite directions (negative correlation), or have no consistent relationship. A value of +1 means they always move together; −1 means they always move in opposite directions; 0 means no linear relationship.

Persistence Analysis

How Sticky Is Each Metric Day-to-Day?

Some health signals are persistent: if your stress is high today, it tends to stay high tomorrow. Others bounce around more freely. An AR(1) coefficient quantifies this "stickiness" — how much of today's value carries over into tomorrow's. A value near 1.0 means the metric barely changes day-to-day; near 0 means each day is essentially independent.

State Transitions

How Does Recovery State Change Day-to-Day?

A Markov transition matrix captures the probability of moving from one "state" to another. Here, we categorize each day's body battery into four states — Low, Moderate, Good, and High — and count how often each transition occurs. This reveals whether recovery tends to be stable, or whether it swings unpredictably.

Each row in the matrix shows: given that today is in state X, what fraction of the time does tomorrow land in each state? High diagonal values mean the system is stable. Off-diagonal values show common transitions.

Workout Effects

How Does Training Affect Next-Day Recovery?

One of the most useful questions in health tracking: does today's workout help or hinder tomorrow's recovery? By comparing next-day health metrics on workout days versus rest days, we can estimate the short-term recovery cost (or benefit) of training.
